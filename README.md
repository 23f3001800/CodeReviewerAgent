# CodeSense — AI Code Reviewer

**Live demo:** [yourname-codesense.streamlit.app](https://yourname-codesense.streamlit.app)

Paste any code and get a structured review in seconds — bugs, security
vulnerabilities, quality improvements, and a refactored version.

## What it does

- Detects security vulnerabilities (SQL injection, XSS, hardcoded secrets, OWASP Top 10)
- Finds bugs and logic errors with line-specific citations
- Scores code quality 1–10 with actionable improvements
- Generates a refactored version with inline explanations
- Conversational follow-up — ask questions about your specific code
- GitHub PR review — paste any PR URL and review the diff directly
